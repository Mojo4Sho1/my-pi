# _SEEDS_INDEX.md

## Purpose

Routing file for reusable seed definitions.

## What a seed is

A seed is a reusable bootstrap context pack for initializing a repo, domain, or workflow class.

## When to use a seed

Use seeds during repo setup, onboarding, or domain initialization where expected structure and guidance are needed.

## Routing guidance

Read only the seed definition relevant to the current initialization task.

## Update rule

Update this index when seed definitions are added, removed, renamed, or re-routed.
