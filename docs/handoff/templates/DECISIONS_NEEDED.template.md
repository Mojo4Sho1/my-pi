```
**Blocking:** [task ID(s) that cannot proceed without this answer]
**Context:** [What the agent was doing when it hit this question]
**Question:** [The specific decision needed — frame as a choice, not open-ended]
**Options considered:**
- A: [option] — [tradeoff]
- B: [option] — [tradeoff]
**Resolution:** [filled in by human]
**Resolved date:** [filled in by human]
```
